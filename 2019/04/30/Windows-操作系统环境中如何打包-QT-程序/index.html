Windows 操作系统环境中如何打包 QT 程序

2019-04-30

记得上次打包 QT 程序还是在几个月前了，今天我把相应的机器人程序按照算法那边的同事要求初步设计完成，又遇到 QT 程序打包的问题。记性不好，特意在此记录下来，以便后续方便查阅之用，由于算法的同事使用的是操作系统环境为 Windows ，而我开发的环境是 Ubuntu ，所以，我需要在 Windows 环境中把 QT 程序打包成可执行的 exe 文件，具体的操作步骤如下：
QT 程序打包前的准备：

  • 操作系统环境：Windows 7 专业版 64 位
  • QT IDE: QT Creator 4.8.0(Enterprise)
  • DLL 查询软件： ProcessExplorer
  • QT 程序打包软件: Enigma Virtual Box

步骤一：设置 QT 程序的构建方式为 release

当 QT 程序编写完成之后，需要进行打包发布程序之前，启动 QT 程序都要使用 release 的构建方式，切记！

步骤二：查找 QT 程序构建成功之后的 release 文件夹

步骤三：使用 ProcessExplorer 查询软件查找运行 QT 执行程序时使用的外部库，并拷贝至步骤二中的 release 文件夹

步骤四：使用 Enigma Virtual Box 打包 QT 程序，并发布程序
